 Overview
This is the frontend of the **Library Management System** built using **React.js**.  
It supports **Admin** and **User** dashboards with role-based features.  
The frontend communicates with the backend using **traditional Axios** for API requests.

------------------------------------------
 Tech Stack
- React.js
- React Router DOM
- Axios
- CSS
- JWT Authentication (via backend)

------------------------------------------

=> User Dashboard

- Search books by title / author
- Request to borrow books
- Reserve books
- View borrowed books
- Check due dates

------------------------------------------

=> Admin Dashboard
Admins can:

- View all books

- Approve or reject borrow requests

- Track issued & returned books
